# Quick Start Guide

## Prerequisites
- **Windows 10/11**
- **Microphone**

## Installation

1.  **Download the Installer** from the [Releases page](https://github.com/your-repo/diktate/releases).
2.  **Run `diktate-setup.exe`**.
3.  The installer will check for **Ollama**.
    - If Ollama is not found, the installer will automatically download and install it for you.
    - If Ollama is found, it will proceed with the installation.
4.  Launch **dIKtate** from your desktop or start menu.

## Usage

### Dictation
1.  Ensure dIKtate is running (check the system tray).
    > *[Add Image of System Tray Icon here]*
2.  Place your cursor in any text field (Word, Notepad, Browser).
3.  Press **`Ctrl+Alt+D`** to start recording.
    > *[Add Image of Recording Overlay/Status Window here]*
4.  Speak naturally.
5.  Press **`Ctrl+Alt+D`** again to stop.
6.  Wait a moment for the text to appear.

### Ask Mode
1.  Press **`Ctrl+Alt+A`**.
    > *[Add Image of Ask Mode UI here]*
2.  Ask a question (e.g., "What is the capital of France?").
3.  Press **`Ctrl+Alt+A`** to stop.
4.  The answer will be copied to your clipboard (or typed, depending on settings).
